TEMPORIZADOR DE APAGADO WINDOWS v2.0

Descripción
Aplicación ejecutable para Windows que permite programar el apagado automático del equipo después de un tiempo determinado. Programa convertido a EXE para fácil ejecución sin necesidad de PowerShell.

Características
✅ Aplicación ejecutable (.exe) - No requiere PowerShell
✅ Interfaz gráfica moderna y colorida
✅ 10 opciones predefinidas de tiempo (5 a 180 minutos)
✅ Entrada personalizada de minutos
✅ Alerta visual y sonora 1 minuto antes del apagado
✅ Contador en tiempo real
✅ Cancelación en cualquier momento
✅ Apagado inmediato con confirmación
✅ Diseño responsivo y fácil de usar

Requisitos del Sistema
Sistema Operativo: Windows 7/8/10/11 (32 o 64 bits)
.NET Framework: 4.5 o superior (incluido en Windows 10/11)
Espacio en disco: ~5 MB
Permisos: Ejecución como usuario normal

Instalación y Ejecución

Instalación (Portable)
1. Descargue el archivo Temporizador de Apagado Windows.exe
2. Colóquelo en cualquier carpeta (Ej: Escritorio, Documentos)
3. Ejecute haciendo doble clic en el archivo
4. No requiere instalación adicional

Crear Acceso Directo (Opcional)
1. Haga clic derecho en el archivo .exe
2. Seleccione "Crear acceso directo"
3. Arrastre el acceso directo al Escritorio o Barra de tareas
4. Puede asignar un icono personalizado si lo desea

Uso del Programa

Pantalla Principal
[ Título: Temporizador de Apagado Windows v2.0 ]
[ Panel con 10 botones de tiempo predefinido ]
[ Campo para tiempo personalizado ]
[ Estado: Contador en tiempo real ]
[ Botones: Cancelar Apagado | Apagar Ahora ]

Pasos para Usar:
1. Ejecute la aplicación (doble clic en .exe)
2. Seleccione un tiempo:
   - Opción rápida: Haga clic en un botón (5, 10, 15... 180 min)
   - Personalizado: Ingrese minutos y haga clic en "Establecer"
3. Confirme el apagado programado en el mensaje emergente
4. Espere la cuenta regresiva en la barra de estado
5. Alerta: 1 minuto antes aparecerá ventana roja con sonido
6. Cancelar: Puede cancelar en cualquier momento con el botón correspondiente

Funciones Detalladas

Opciones de Tiempo Predefinidas
- Corto plazo: 5, 10, 15 minutos (azul, verde, morado)
- Medio plazo: 30, 45, 60 minutos (naranja, verde, rojo)
- Largo plazo: 90, 120, 150, 180 minutos (azul oscuro, rosa, amarillo, gris)

Tiempo Personalizado
- Ingrese cualquier número entero de minutos
- Mínimo: 1 minuto
- Máximo: Prácticamente ilimitado (solo por precisión del temporizador)
- Haga clic en "Establecer" para activar

Sistema de Alertas
- Confirmación inicial: Muestra hora exacta de apagado
- Alerta crítica (1 minuto): Ventana roja fullscreen con sonido de advertencia
- Actualización continua: Contador que disminuye cada segundo

Controles Principales
- Cancelar Apagado: Detiene temporizador activo (rojo)
- Apagar Ahora: Apagado inmediato con confirmación (naranja)

Solución de Problemas

Problema: No se ejecuta
Soluciones:
1. Verifique que Windows no marque el archivo como peligroso
2. Haga clic derecho → "Ejecutar como administrador" (si hay bloqueo)
3. Verifique que tenga .NET Framework instalado (Windows Update)

Problema: No apaga el equipo
Posibles causas:
1. Otro programa bloquea el apagado
2. Actualizaciones de Windows en proceso
3. Permisos insuficientes

Soluciones:
1. Cierre programas manualmente
2. Ejecute como administrador temporalmente
3. Use "Apagar Ahora" para forzar cierre

Problema: Alerta no aparece
Verifique:
1. El programa está minimizado (la alerta aparece encima)
2. El sonido del sistema está activado
3. No hay otros programas en pantalla completa

Preguntas Frecuentes

¿Puedo usar decimales en tiempo personalizado?
No, solo números enteros (minutos completos).

¿Qué pasa si cierro la ventana principal?
El temporizador se cancela automáticamente.

¿Funciona en Windows Server?
Sí, pero puede requerir .NET Framework adicional.

¿Puedo cambiar el idioma?
Actualmente solo en español.

¿Hay versión de 32 bits?
El mismo ejecutable funciona en ambas arquitecturas.

¿Se puede desinstalar?
Simplemente borre el archivo .exe (no deja residuos).
